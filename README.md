# Java-for-QA-Engineers

This repository contains Java programs and examples designed to support QA engineers, automation testers, and beginners in strengthening their Java programming knowledge.

## Repository Structure

### Basics

Includes examples for variables, data types, operators, loops, and conditional statements.

### OOPs

Includes examples for inheritance, polymorphism, abstraction, encapsulation, and interfaces.

### Collections

Includes examples for ArrayList, HashMap, HashSet, and collection comparisons.

### String Programs

Includes logic-building exercises such as reverse string, palindrome, anagram, duplicate characters, and character count.

### Number Programs

Includes programs for factorial, prime number, Fibonacci series, Armstrong number, swapping numbers, and digit shifting logic.

### Arrays

Includes examples for sorting, largest number, second largest, duplicate elements, and missing number detection.

### Exception Handling

Includes try-catch examples, throw vs throws, and custom exception handling.

### File Handling

Includes reading, writing, and CSV file examples.

### Java 8 Features

Includes lambda expressions, streams, filters, and Optional examples.

## Purpose

This repository is intended to demonstrate:

* Core Java knowledge
* Logic-building skills
* OOP concepts
* Collections and data handling
* Java programs commonly used in automation testing
* Interview preparation for QA and automation roles

These examples are generic, reusable, and suitable for public portfolio use.
