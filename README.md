## 🎯 Purpose

✔️ Strong logic-building skills

✔️ Collections and data handling

---

## 📂 Repository Structure

### 🔹 Basics

Variables, data types, operators, loops, and conditional statements

### 🔹 OOPs

Inheritance, polymorphism, abstraction, encapsulation, interfaces

### 🔹 Collections

ArrayList, HashMap, HashSet, and collection comparisons

### 🔹 String Programs

Reverse string, palindrome, anagram, duplicate characters, character count

### 🔹 Number Programs

Factorial, prime number, Fibonacci, Armstrong number, swapping, digit shifting logic

### 🔹 Arrays

Sorting, largest number, second largest, duplicates, missing number detection

### 🔹 Exception Handling

Try-catch, throw vs throws, custom exceptions

### 🔹 File Handling

File read/write operations, CSV handling

### 🔹 Java 8 Features

Lambda expressions, streams, filters, Optional
